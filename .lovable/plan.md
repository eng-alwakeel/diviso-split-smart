

# تعديل حجم اللوقو في البار العلوي

## المشكلة
اللوقو في البار العلوي (Header) أصغر بكثير من اللوقو في صفحة تسجيل الدخول.

| الموقع | الحجم الحالي | الحجم بالبيكسل |
|--------|-------------|---------------|
| البار العلوي (Header) | `h-6 sm:h-8` | 24px موبايل / 32px ديسكتوب |
| صفحة تسجيل الدخول | `h-10` | 40px |

## الحل

تكبير اللوقو في البار العلوي ليتناسب مع صفحة تسجيل الدخول.

### الملفات المطلوب تعديلها:

**1. `src/components/AppHeader.tsx`** (البار العلوي للداشبورد وصفحة Auth)
- تغيير حجم اللوقو من `h-6 sm:h-8` الى `h-8 sm:h-10`
- تحديث container height من `h-8` الى `h-10`

**2. `src/components/Header.tsx`** (البار العلوي للصفحة الرئيسية)
- تغيير حجم اللوقو من `h-6 sm:h-8` الى `h-8 sm:h-10`
- تحديث container height من `h-8` الى `h-10`

### النتيجة المتوقعة

| الموقع | قبل | بعد |
|--------|-----|-----|
| البار العلوي (موبايل) | 24px | 32px |
| البار العلوي (ديسكتوب) | 32px | 40px |
| صفحة تسجيل الدخول | 40px | 40px (بدون تغيير) |

### تفاصيل تقنية

التغيير في `AppHeader.tsx` سطر 208:
```
// قبل
className="h-6 sm:h-8 w-auto"
// بعد
className="h-8 sm:h-10 w-auto"
```

التغيير في `Header.tsx` سطر 67:
```
// قبل
className="h-6 sm:h-8 w-auto"
// بعد
className="h-8 sm:h-10 w-auto"
```

بالاضافة لتحديث اللوقو في القائمة الجانبية (Mobile Menu) في `Header.tsx` سطر 110.
